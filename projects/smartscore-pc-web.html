PC WEB 골프장/부킹 서비스 개발

앱에서 제공하던 골프장/부킹 서비스를 PC 웹에서도 사용할 수 있도록 골프장 영역과 공통 컴포넌트를 개발했습니다. 검색/필터 UI를 공통화하고, Kakao 지도 API를 활용해 골프장 목록과 지도 이동이 연결되는 탐색 흐름을 구현했습니다.

PHP Vue.js MySQL JavaScript Kakao Map API
Period
2020.09 - 2021.02
Role
PC 웹 골프장 영역 개발, 공통 컴포넌트 개발, 백엔드 연계
Focus
골프장 화면, 검색/필터, 지도 API, 공통 UI, 데이터 연계

문제

기존 앱 중심으로 제공되던 골프장과 부킹 서비스를 PC 웹에서도 사용할 수 있도록 확장해야 했습니다. 사용자가 다른 환경에서도 일관된 흐름으로 서비스를 이용할 수 있는 웹 화면과 공통 구조가 필요했고, 골프장 목록, 검색/필터, 지도 기반 위치 탐색이 자연스럽게 이어져야 했습니다.

기여

  • PC WEB 골프장 영역과 부킹 서비스 공통 영역 개발
  • Vue.js 기반 화면 개편과 공통 컴포넌트 개발
  • 검색/필터 UI를 공통 컴포넌트화해 화면 간 일관성과 재사용성 확보
  • Kakao 지도 API를 활용한 골프장 위치 표시와 목록 선택 연동 구현
  • 앱 서비스의 웹 확장 흐름에 맞춘 화면 및 데이터 연계
  • Redmine, Git 기반 협업과 이슈 관리

대표 구현 기능

  • 칩 형태의 골프장 목록을 선택하면 해당 골프장 위치로 지도가 이동하는 Kakao 지도 API 연동 기능을 구현했습니다.
  • 검색 조건과 필터 UI를 공통 컴포넌트로 정리해 골프장/부킹 화면에서 반복되는 구현을 줄였습니다.
  • PC 웹 사용자가 골프장 정보를 탐색할 수 있도록 목록, 검색, 필터, 지도 이동 흐름을 연결했습니다.

담당 범위

  • 앱 중심으로 제공되던 기능을 PC 웹 사용자가 이해할 수 있는 화면 흐름으로 재구성했습니다.
  • 골프장 관련 화면의 목록, 검색, 필터, 지도 탐색 흐름을 중심으로 작업했습니다.
  • 공통 레이아웃과 반복 사용 영역을 기준으로 Vue.js 컴포넌트 구조를 잡았습니다.
  • 목록, 상세, 검색 조건, 지도 위치처럼 사용자 화면에 필요한 데이터를 기존 백엔드 흐름과 연결했습니다.
  • 이슈 단위로 작업 범위를 나누고, 화면 동작과 데이터 표시가 함께 맞는지 확인했습니다.

결과

앱에서 제공하던 핵심 서비스를 PC 웹으로 확장하면서 사용자 접점을 넓혔고, 골프장과 부킹 서비스의 공통 웹 영역을 구성했습니다. 골프장 목록, 검색/필터, 지도 이동이 연결된 탐색 경험을 구현했고, 기존 서비스 데이터 흐름을 이해한 상태에서 웹 화면과 백엔드 연계를 함께 다뤘습니다.
