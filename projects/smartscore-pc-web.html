문제

기여

  • PC WEB 골프장/부킹 서비스 공통 영역 개발
  • Vue.js 기반 화면 개편과 공통 레이아웃 작업
  • 앱 서비스의 웹 확장 흐름에 맞춘 화면 및 데이터 연계
  • Redmine, Git 기반 협업과 이슈 관리

결과

앱에서 제공하던 핵심 서비스를 PC 웹으로 확장하면서 사용자 접점을 넓혔고, 골프장과 부킹 서비스의 공통 웹 영역을 구성했습니다.
