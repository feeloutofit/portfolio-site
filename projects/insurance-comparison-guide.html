보험/수수료 데이터 처리 및 보험료 산출 시스템

보험사별 실적 데이터를 전문 형태로 수신해 파싱하고 Oracle DB에 적재한 뒤, 매니저가 실적 확인과 수수료 계산에 활용할 수 있는 업무 흐름을 개발했습니다. 자동차 보험 수수료, 수금수수료, 보험료 표 가공, 자동차/일반 보험료 산출 로직을 다뤘습니다.

PHP CodeIgniter Zend Framework Oracle FTP Excel
Period
2015.04 - 2019.12 재직 중 수행
Role
웹 개발, 데이터 처리, 수수료/보험료 계산 로직 개발
Focus
실적 데이터 파싱, DB 적재, 수수료 계산, 보험료 산출

문제

보험사별로 전달되는 실적 데이터와 보험료 표를 내부 업무 시스템에서 활용 가능한 형태로 가공해야 했습니다. 매니저가 실적을 확인하고, 계약 유지나 보험사별 지급 조건에 따라 수수료를 계산할 수 있는 흐름이 필요했습니다.

기여

  • 보험사별 실적 데이터를 전문 형태로 수신하고 파싱해 Oracle DB에 적재하는 흐름 구현
  • 매니저가 적재된 실적을 확인하고 수수료 계산에 활용할 수 있는 조회/처리 흐름 개발
  • 자동차 보험 수수료와 수금수수료 계산 프로그램 개발
  • 보험사 기본 보험료 표 Excel 데이터를 가공해 자동차보험 자동 계산 프로그램 유지보수
  • 일반보험료 관련 보험사 데이터를 기반으로 보험료 산출 프로그램 개발
  • PHP, CodeIgniter/Zend Framework, Oracle 기반 데이터 처리 로직과 업무 화면 구현

대표 구현 기능

  • 보험사 실적 전문 데이터를 내부 DB 구조에 맞게 파싱하고 적재했습니다.
  • 실적과 계약 유지 조건을 바탕으로 자동차 보험 수수료와 수금수수료를 계산하는 업무 로직을 구현했습니다.
  • 보험사 기본 보험료 표를 Excel로 받아 가공하고, 자동차/일반 보험료 산출 흐름을 유지보수·개발했습니다.

담당 범위

  • 외부 보험사 데이터의 수신 방식과 데이터 구조를 확인하고 파싱 기준을 정리했습니다.
  • 전문 형태의 실적 데이터를 Oracle DB에 적재해 내부 업무 시스템에서 조회할 수 있도록 처리했습니다.
  • 매니저가 실적을 확인하고 수수료 계산 결과를 업무에 활용할 수 있는 화면/로직을 개발했습니다.
  • 자동차 보험 수수료, 수금수수료, 자동차 보험료 자동 계산, 일반보험료 산출처럼 금액 계산이 중요한 기능을 다뤘습니다.
  • 외부 데이터 수신부터 DB 저장, 조회, 계산까지 하나의 업무 흐름으로 이어지도록 처리 구간을 구성했습니다.

결과

보험사에서 전달되는 외부 데이터를 내부 업무 시스템에서 활용 가능한 형태로 가공했고, 매니저가 실적을 확인하고 수수료 계산에 활용할 수 있는 업무 흐름을 시스템화했습니다. 자동차 보험 수수료, 수금수수료, 자동차/일반 보험료 산출처럼 보험 도메인의 복잡한 계산 규칙을 구현했습니다.
