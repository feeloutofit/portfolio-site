보험비교안내표 홈페이지 개발

보험대리점 협회 데이터를 FTP로 가져와 파싱하고 Oracle DB에 저장한 뒤, 상품 설정과 금액 기준에 따라 비교군을 조회할 수 있는 홈페이지를 개발했습니다.

PHP CodeIgniter Oracle FTP Linux
Period
2018.03 - 2018.04
Role
데이터 처리 및 웹 기능 개발
Focus
FTP 수집, 파싱, DB 저장, 조건별 비교 조회

문제

외부 협회에서 제공하는 보험 상품 데이터를 내부 서비스에서 활용할 수 있도록 수집하고, 비교 기준에 맞게 조회 가능한 형태로 정리해야 했습니다.

기여

  • FTP 기반 외부 데이터 수집 흐름 구현
  • 수집 데이터 파싱과 Oracle DB 저장 처리
  • 상품 설정과 금액 기준에 따른 비교군 조회 화면 개발
  • PHP, CodeIgniter 기반 웹 화면과 데이터 처리 로직 구현

담당 범위

  • 외부에서 제공하는 파일의 수집 방식과 데이터 구조를 확인하고 파싱 기준을 정리했습니다.
  • 수집한 데이터를 Oracle DB에 저장해 업무 시스템에서 다시 조회할 수 있는 흐름을 구현했습니다.
  • 상품 조건과 금액 기준을 바탕으로 비교군을 조회하는 로직과 화면을 연결했습니다.
  • 데이터 수집부터 화면 출력까지 하나의 업무 흐름으로 이어지도록 처리 구간을 구성했습니다.

결과

외부 데이터를 업무 시스템에서 활용 가능한 형태로 가공했고, 보험 상품 비교 정보를 조회할 수 있는 홈페이지를 제공했습니다. 사람이 파일을 받아 수작업으로 확인하던 데이터를 시스템 안에서 저장, 조회, 비교할 수 있는 형태로 연결했습니다.
